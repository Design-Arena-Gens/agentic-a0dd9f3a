# Attach Document Guide

A lightweight Next.js site that explains how to upload and share documents in a chat-style interface across desktop and mobile devices.

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS modules via global stylesheet

## Scripts

- `npm run dev` – start the development server
- `npm run build` – create an optimized production build
- `npm start` – serve the build output
- `npm run lint` – run ESLint with the Next.js config

## Local Development

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deployment

The project is configured to deploy seamlessly to Vercel. Run `npm run build` locally to verify, then push or deploy using the Vercel CLI.
